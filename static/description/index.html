Custom Asset Management v18.0

A comprehensive, standalone Asset Management module for Odoo 18. Completely independent from Odoo's built-in Accounting Asset module.

Models Included

ModelDescription
asset.companyCompanies for asset ownership tracking
asset.divisionOrganisational divisions
asset.districtGeographic districts
asset.departmentDepartments responsible for assets
asset.unitOperational or organisational units
asset.designationJob designations linked to assets
asset.tagTagging/labelling for quick filtering
asset.brandManufacturer / brand master
asset.serviceService types for maintenance
asset.accessoryAccessory items for assets
asset.financial.yearFiscal year definitions with date validation
asset.colorColor coding for physical identification
asset.insurance.partnerInsurance companies with contact details
asset.classTop-level asset classification
asset.categoryMid-level classification (under Class)
asset.itemGranular item classification (under Category)
asset.office.locationPhysical office/building locations
asset.floor.locationFloor-level locations within an office

Key Features

  • Fully independent from Odoo Accounting asset module
  • Archive/unarchive support on every model
  • Smart stat buttons with drill-down navigation
  • Hierarchical classification: Class → Category → Item
  • Hierarchical location: Office → Floor
  • Date validation on Financial Year
  • Email validation on Insurance Partner
  • SQL unique constraints on all name fields
  • Full search, group-by and filter support on every model
  • Odoo 18 compatible view syntax throughout
